질문 3
다음 중 Kotlin 및 Java 코드에 대한 설명으로 옳지 않은 것은?

▢ Kotlin 코드와 Java 코드를 나란히 실행할 수 있습니다.

▢ 기존 자바 프로그램에 Kotlin 코드를 추가할 수 있습니다.

▢ 기존 Java 코드를 Kotlin으로 마이그레이션할 수 있습니다.

▢ Kotlin 코드는 Java 코드보다 빠르게 실행됩니다.

ans) 4번 코틀린 언어와 자바는 언어는 서로 사용이 가능합니다. 즉, 4번과 같은 내용은 언급이 없었고, 같이 사용이 가능한만큼 빠르게 실행되는 것이 아닌 같이 실행됩니다.
